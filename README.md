# borradez

Projeto para calcular um fator de borradez em imagens e comparar coleções.

Estrutura:
- `teste-01.ipynb`: notebook principal com função `calcular_fator_borradez` e célula de teste.
- `imagens_Boas/`, `imagens_Ruins/`: pastas de imagens usadas para comparação.

Instruções rápidas:
1. Abra o notebook `teste-01.ipynb` e execute as células na ordem.
2. Ajuste as pastas ou parâmetros conforme necessário.
